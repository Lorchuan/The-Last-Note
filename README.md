# The Last Note 《尾声》

这是一个超酷的 3D PC/VR 游戏项目！玩家通过特制的手套画符释放技能击败怪物，逐渐揭开文明退行的故事。

## ✨ 游戏特色
* **沉浸式体验**：佩戴手套画符施法，剧情采用环境叙事。

## 🕹️ 操作说明
* **左手柄或WASD**：控制移动和视角转动
* **右手食指/鼠标左键**：施法

## 🛠️ 开发环境
* **引擎**：Tuanjie 1.6.4
* **目标平台**：PC
